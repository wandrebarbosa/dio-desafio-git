comandos git

  • git init - Inicia e instala as funcionalidades do git no repositorio
  • git status - Verifica toda alteração e informa arquivos editados, prontos e disponiveis para subir para github
  • git add . - Adiciona todos os elementos modificados para prontos, para enfim ser levados para o Github
  • git clone "link do repositorio" - Clona um repositorio direto do GitHub para sua maquina
  • git commit -m "mensagem do commit" - Marca as alterações com uma mensagem, tipo um "save game" que pode ser restaurada a partir de um commit anterior
  • git push origin master - sobe todas as alterações "commitadas" com o código anterior, para definitivamente enviar para o Github, podendo escolher a branch (linha do tempo), neste caso a master
